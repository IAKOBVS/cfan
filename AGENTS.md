# cfan

## Key files

| File | Role |
|------|------|
| `cfan.c` | Main loop: reads temps, picks fan speed from curve, writes PWM |
| `cfan-print.c` | Visualizes active fan curve |
| `config.h` / `config.mk` | User-local config (gitignored, copied from `config.def.*`) |
| `table-temp.h` | User-local temp sensor list (gitignored, copied from `table-temp.def.h`) |
| `table-temp.def.h` | Default sensor config (CPU only) |
| `config.def.h` | Fan curves (`c_table_temptospeed_med`, `c_table_temptospeed_high`), timing constants |
| `cpu.generated.h` | Auto-detected CPU temp file path (run-time generated by `getcpufile`) |
| `table-fans.generated.h` | Auto-detected PWM device paths (run-time generated by `getpwmfiles`) |
| `cfan-set-pwm` | Shell script to manually set PWM speed (not compiled) |
| `defoverride` | Copies current `config.h`/`table-temp.h` back to `.def.h` defaults |

## Config quirks

- **CUDA** is `#define`d in `config.def.h` but link flags are commented out in `config.def.mk`. Run `make enable-cuda` to uncomment both, or `make disable-cuda` to undo.
- **Sysfs paths MUST start with `/sys/devices/platform/`** (not `/sys/class/`). The `path_sysfs_resolve` function globs the numeric hwmon/thermal suffix that may change across reboots. Use `realpath` to resolve a `/sys/class/` link to its platform path.
- `defoverride` saves your running `config.h`/`table-temp.h` back to the `.def.h` originals â€” run it when you want to preserve changes for git.

## Testing & linting

- No test suite exists.
- The Makefile compiles with `-fanalyzer -Warray-bounds -Wnull-dereference -Wformat -Wunused -Wwrite-strings`. These serve as the linting step.
- No formatter config (no `.clang-format`). Keep style consistent with existing files.

## Architecture notes

- `c_table_temps[]` â€” file paths for temperature sensors
- `c_table_fn_temps[]` â€” function pointers for temperature reading (e.g. NVML GPU temps)
- `c_table_fans[]` / `c_table_fans_enable[]` â€” PWM device files (generated)
- `temptospeed` â€” pointer to active fan curve (default, medium, or high)
- On exit, fans are restored to `PWM_ENABLE_AUTO` (kernel auto mode)
- Lock file at `/tmp/cfan/cfan.lock` prevents multiple instances
